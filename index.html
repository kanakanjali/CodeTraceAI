Project architecture

          
Built like a real developer tool, not a generic dashboard.

        


        

          

            01
            
Recorder hooks

            

              Git hooks, file watchers, and shell wrappers capture edits,
              commands, package installs, and checkpoints during an AI coding
              session.
            

          

          

            02
            
Risk engine

            

              Rules score secrets exposure, risky command patterns, unsafe
              dependency drift, auth changes, and sensitive file access.
            

          

          

            03
            
Trace graph

            

              Events are linked into a timeline so reviewers can replay what the
              AI changed and understand which files may need tests.
            

          

          

            04
            
Rollback report

            

              The final report includes a session score, risky diffs, command
              log, safe checkpoint, and suggested verification steps.
